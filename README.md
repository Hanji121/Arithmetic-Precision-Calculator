# Arithmetic Precision Calculator

## Overview
Arithmetic Precision Calculator is a C-based application designed to perform arithmetic operations with high precision and accuracy. The project demonstrates concepts such as file handling, modular programming, data processing, and arithmetic computation.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Precision-based calculations
- User-friendly menu-driven interface
- Error handling for invalid inputs

## Technologies Used
- C Programming
- GCC Compiler
- Git & GitHub
